# ISP-Sync Kanban

Sistema de coordinación operativa para ISP (Atención al Cliente, Planta Externa, Datacenter).

## Estructura del Proyecto
- **/backend**: API REST desarrollada en Python con FastAPI.
- **/frontend**: Interfaz de usuario (SPA) desarrollada en React + Mantine UI.

## Requisitos Previos
- Docker y Docker Compose
- Node.js 18+
- Python 3.11+
